## Description
Изучение [Nest](https://github.com/nestjs/nest) по [виде лекции](https://coursehunter.net/course/nestjs-s-nulya-sovremennyy-backend-na-typescript-i-node-js) ([git](https://github.com/AlariCode/top-api-demo)).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
